# 总结：

功能分为：

- 核心业务：比如登录，增加数据，删除数据
- 周边功能：比如性能统计，日志，事务管理

周边功能被定义为切面

## aop的目的

切入点（Pointcut）
 在哪些类，哪些方法上切入（**where**）

通知（Advice）
 在方法执行的什么实际（**when:**方法前/方法后/方法前后）做什么（**what:**增强的功能）

切面（Aspect）
 切面 = 切入点 + 通知，通俗点就是：**在什么时机，什么地方，做什么增强！** 

织入（Weaving）
 把切面加入到对象，并创建出代理对象的过程。（由 Spring 来完成）

aop有两种实现方式：

* jdk动态代理：主要用在接口上面，没接口不行
* cglib动态代理：可以用在没接口的类上
